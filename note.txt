///// Шаг первый. /////
 Создаем папку server и папуку config. 
config содержет файл db.js - где мы потключаем локальную базу данных mongoDB.
создаем папку pages с файлом router.js, который будет содержать роутинг всех основных страниц.

///// Шаг второй. /////
В папке server создаем папку Categories 
В папке Сategories мы создаем коллекцию (схему), 
для категорий в файле Categories.js 
задаем структуру данных категорий, в БД. 

В файле router.js строим маршрут для получения категорий.
Для этого подключаемя exspress импортируем Categories.
Создаем маршрут с запросом GET - запрос на /api/category.
Данный router.js подключаем в основной файл index.js 

создаем файл seed.js с переменной, которая хранит в себе информацию о категориях
в нашем случае это наименование категорий. 
так же в этом файле будет написана функция, которая записывает эти днанные в ДБ.
